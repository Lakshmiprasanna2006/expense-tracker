# Expense Tracker

A simple web-based Expense Tracker application that allows users to:

- Add income and expense transactions
- Delete transactions
- Automatically update total income, total expense, and balance
- Store transactions using Local Storage

## Technologies Used
HTML
CSS
JavaScript
Bootstrap

## Features
Add Transactions
Delete Transactions
Automatic Balance Calculation
Persistent Data with Local Storage

## How to Run

1. Download the repository
2. Open home.html in a browser
3. Start tracking your expenses
